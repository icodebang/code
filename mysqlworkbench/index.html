设计

MySQL Workbench使DBA，开发人员或数据架构师能够可视化地设计，建模，生成和管理数据库。它包括数据建模人员创建复杂的ER模型，进行正向和反向工程所需的一切，还提供了用于执行通常需要大量时间和精力的困难的变更管理和文档编制任务的关键功能。

开发

MySQL Workbench提供了用于创建，执行和优化SQL查询的可视化工具。SQL编辑器提供了高亮的颜色语法，自动完成功能，SQL代码段的重用以及SQL的执行历史记录。数据库连接面板使开发人员可以轻松管理标准数据库连接，包括MySQL Fabric。对象浏览器提供对数据库架构和对象的即时访问。

管理

MySQL Workbench提供了一个可视控制台，可以轻松地管理MySQL环境并更好地查看数据库。开发人员和DBA可以使用可视化工具来配置服务器，管理用户，执行备份和恢复，检查审核数据以及查看数据库运行状况。

视觉效果仪表板

MySQL Workbench提供了一套工具来改善MySQL应用程序的性能。DBA可以使用性能仪表板快速查看关键性能指标。通过性能报告，可以轻松识别和访问IO热点，高成本SQL语句等。另外，只需单击一下，开发人员就可以看到在哪里可以使用经过改进且易于使用的Visual Explain Plan优化查询。

数据库迁移

MySQL Workbench现在提供了一个完整，易于使用的解决方案，用于将Microsoft SQL Server，Microsoft Access，Sybase ASE，PostreSQL和其他RDBMS表，对象和数据迁移到MySQL。开发人员和DBA可以快速轻松地将现有应用程序转换为在Windows和其他平台上的MySQL上运行。迁移还支持从早期版本的MySQL迁移到最新版本。
